Black Milord

Black Milord is an free open source eBook editor. It is desired to create or edit existing eBooks in mobi format.

It's main features are:
  • Import and saving eBooks in mobi format
  • Redo/Undo
  • Find and Replace
  • Book's metadata editor
  • Spell checking (require Aspell and dictionaries installed)
  • Highlighting spelling errors and HTML tags
